# Weave Notes App

Electron + React + TypeScript + Tailwind + Zustand starter.

## Commands

- `npm install` - install dependencies
- `npm run dev` - start Vite and Electron together
- `npm run build` - build the renderer app
- `npm run preview` - preview the renderer locally

## Project structure

- `electron/` - Electron main and preload code
- `src/` - React app and Zustand store
- `public/` - Vite static entry point
